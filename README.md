# Password Generator

**Live Demo:** [https://password-gen-v.netlify.app/](https://password-gen-v.netlify.app/)

## Overview

A simple and interactive **React-based password generator** that allows users to create secure passwords based on customizable criteria.

Users can control the password length and choose whether to include numbers and special characters, then instantly generate and copy a password.

## Features

* Dynamic password generation
* Adjustable password length
* Optional number inclusion
* Optional special character inclusion
* One-click copy to clipboard
* Responsive UI

## Tech Stack

* React
* JavaScript
* Tailwind CSS
* Vite

## Getting Started

Follow the steps below to run the project locally.

1. **Clone the repository**

   ```bash
   git clone https://github.com/denichinv/password-generator.git
   ```

2. **Navigate into the project directory**

   ```bash
   cd password-generator
   ```

3. **Install dependencies**

   ```bash
   npm install
   ```

4. **Start the development server**

   ```bash
   npm start
   ```

The application will run at `http://localhost:3000`.

## Usage

1. Adjust the password length using the slider
2. Enable or disable numbers and special characters
3. Click **Generate** to create a password
4. Click the copy button to save it to your clipboard

## Customization

You can extend or modify the password logic by editing the password generation function in the source code. UI styling can be adjusted using Tailwind CSS utility classes.
